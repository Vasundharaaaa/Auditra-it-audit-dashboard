💡 Recommendations

🟡 Is all stored data password protected?
Enforce strong password policies (min 12 chars, alphanumeric+symbols). Encrypt all stored personal data immediately.
🟡 Are system passwords changed regularly (every 90 days)?
Set up mandatory 90-day password rotation. Use a password manager. Enable MFA where possible.
🟡 Is antivirus software installed and up-to-date on all devices?
Deploy reputable endpoint protection (Windows Defender, Malwarebytes, or equivalent) on ALL devices. Schedule weekly scans.
🟡 Are operating systems and software regularly updated/patched?
Enable automatic OS and software updates. Review pending patches weekly. Prioritize critical/security patches.
🔴 Do you take regular automated backups of important data?
Implement 3-2-1 backup: 3 copies, 2 media types (local + cloud), 1 offsite. Test restores quarterly.
🟡 Is CCTV footage access restricted and stored securely?
Restrict CCTV system access to authorized personnel only. Log all access. Encrypt stored footage. Review retention policy.
🟡 Are USB / removable devices controlled or restricted?
Deploy endpoint management to block unauthorized USB devices. Use Group Policy or dedicated DLP tools.
🔴 Is sensitive data shared securely (not via WhatsApp/personal email)?
Stop using WhatsApp/personal email for any organizational data. Use encrypted channels: Signal for Business, email with TLS, or secure file-sharing platforms.
🟡 Is sensitive/personal data encrypted at rest and in transit?
Encrypt all sensitive files with AES-256. Use BitLocker (Windows) or FileVault (Mac) for disk encryption. Use TLS 1.3 for all data transmission.
🔴 Is there a documented incident response/breach plan?
Create a basic Incident Response Plan covering: detection, containment, eradication, recovery, and CERT-In notification within 6 hours of a cybersecurity incident.
🔴 Are devices automatically locked after inactivity?
Configure all devices to auto-lock after 5 minutes of inactivity. Enforce via OS settings or Group Policy.
