# 🛡️ AUDITRA

### *Intelligent IT Audit & Compliance System for Micro Organizations*

---

## 📌 Overview

**AUDITRA** is a multi-layered, interactive IT audit dashboard designed to simplify security and compliance assessments for small organizations such as schools, clinics, and startups.

This project demonstrates a **real-world IT Infrastructure and Compliance Audit** conducted at:

🏫 **Amirthaa Kids Playschool, Karaikudi, Tamil Nadu**

---

## 🚀 Key Features

✨ Interactive audit questionnaire (step-by-step)

📊 Automated risk scoring system

📈 Advanced visual analytics (Pie, Bar, Line, Radar charts)

📄 Auto-generated professional audit reports (PDF)

📸 Geo-tagged audit evidence upload

🗂️ Audit history tracking using local storage

🎯 Clean, modern dashboard UI with dark theme


---

## 🧠 How It Works

1. User selects audit details (organization, location, date)
2. System collects responses through structured questions
3. Each response is scored based on risk weight
4. Data is processed to generate:

   * Risk Score
   * Findings
   * Visual insights
   * Recommendations
5. A detailed audit report is generated automatically
